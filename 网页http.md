# DNS
- 使用域名代替记忆ip地址来访问服务器
	- 有两种方式，一种迭代查询与本地服务器交互（有高速缓存）
	- 一种递归查询，与根服务器交互，递归查询，比较慢
	- 有了ip可以请求，浏览器和主机也会缓存ip
- 如果明文tcp传输，有人可以监控
	- 对称加密
	- 非对称加密：无法解决中间人攻击
	- CA机构可以帮忙肉身跑去运输公钥和私钥
	- http做了tls加密，非常复杂，就变成https加密通信
	- http/2是最常用的，http/3已经加入了很多加密内容
